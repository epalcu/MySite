  • FPGA Music Box

    Aug 2014 – Dec 2014

    The focus of this project was to transform an FPGA Basys board into a music box through VHDL. It gives users the ability to play preset songs, as well as play songs themselves. Along with playing songs, the music box comes with fun features that indicate the notes being played, whether on a monitor or across the 7- segment display. These songs are implemented using sequential logic in VHDL to alleviate memory for the storing of displayable notes. The displaying of the notes on the monitor was achieved by implementing a bitmap and driving the figure through the Basys board's VGA capabilities. The displaying of the notes across the 7-segment display was achieved by setting a simple counter that would trigger the corresponding pin for each note.

  • Research Robotic Arm

    Aug 2015 – Dec 2015

    The goal of this Senior Design project was to revive an early 1990s, seven joint Research Robotic Arm. Due to time constraints, only two of the seven joints were completed. Project responsibilities included taking lead in motor controller research, developing a graphical user interface for allowing real-time control of the brushless motors, and developing code that allowed communication between host-interface and each joint controller. Furthermore, providing a proposal for interlinking all seven joint controllers across a CAN-open protocol for controlling the arm.

  • Work Scheduling Manager

    Jan 2016 – May 2016

    The aim of this Software Engineering project was to design a superior lab scheduling system to that of the OIT lab services system currently facilitated by UTK. The major improvements pertain to enforcing various constraints, allowing users to view scheduling specifics, and providing a mobile compatible website. These functionalities were accomplished by utilizing HTML and Bootstrap framework for the front-end design, python and Flask for back-end and developing environment, and MySQL and SQLAlchemy for the necessary database. Personal responsibilities included the entire development of the administrator view, the implementation of the schedule viewing through a calendar, the general security of the site, and lead report writer/ presenter.

  • Parallelized Facial Recognition

    Aug 2016 – Dec 2016

    The objective behind of this project was to develop facial recognition software for the new HPC clusters developed at UTK: BOB and ALICE. By utilizing the OpenCV framework, the facial recognition was parallelized across both clusters to achieve efficient image processing. This allowed large image datasets, consisting of 500 images, to be processed concurrently across the nodes in a fraction of the time. Furthermore, by leveraging machine-learning algorithms and techniques, parallelized facial prediction was also achieved. This included functionality extends each HPC's potential, by allowing users to log in via facial recognition as opposed to the traditional username/password approach.
